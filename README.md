# Task 04: Simple Keylogger
Created during my Internship at **Prodigy InfoTech**.

## Description
This project is a basic keylogger written in Python that records keyboard inputs and saves them to a local text file. 

## Features
- Captures alphanumeric keys and special keys.
- Timestamps for each entry.
- Clean stop mechanism using the 'Esc' key.

## Ethical Note
**Warning:** This project is for educational purposes only. Unauthorized use of keyloggers is illegal. This was developed to understand the mechanics of keyboard hooking and data logging in a controlled environment.

## Requirements
- Python 3.x
- `pynput` library
